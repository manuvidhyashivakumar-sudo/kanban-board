# Kanban Board

A modern Kanban Task Management Application built using React JS and TailwindCSS.

## Features

- Create tasks
- Edit tasks
- Delete tasks
- Drag and drop tasks
- Task detail modal
- localStorage persistence
- Responsive UI
- Priority and tags

## Tech Stack

- React JS
- TailwindCSS
- dnd-kit
- Context API
- localStorage

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Deployed using Netlify.
